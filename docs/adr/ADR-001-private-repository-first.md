# ADR-001: Private Repository First

## Status

Accepted

## Context

The Local AI Platform is in its initial foundation and architecture phase. Early development requires room for iteration, review, and correction before public release expectations are introduced.

## Decision

The repository will remain private during initial development. It may be made public after the foundation, governance, architecture, and initial platform direction are stable enough for external review.

## Consequences

This allows the project to develop with disciplined iteration before public exposure. It also requires future review before publication to ensure documentation quality, repository hygiene, and sensitive information handling are appropriate for a public GitHub repository.
