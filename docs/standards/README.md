# Engineering Standards

## Purpose

Engineering Standards define repeatable expectations for how this repository is designed, documented, and maintained. They provide durable guidance for work that should be consistent across the platform.

## Standards and ADRs

Standards describe ongoing engineering practices. Architecture Decision Records document specific decisions made at a point in time, including their context and consequences.

## Evolution

Standards may evolve as the platform matures. Changes should be reviewed, documented, and applied consistently to future work.
